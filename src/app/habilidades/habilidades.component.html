Perfil Profesional

Me especializo en el desarrollo frontend con Angular y backend con Java y Node.js, aplicando buenas prácticas como SOLID y metodologías ágiles como Scrum. Mis Habilidades Técnicas

Angular

Con más de 4 años de experiencia en desarrollo frontend, uso Angular para crear aplicaciones web dinámicas, reactivas y bien estructuradas. He trabajado en proyectos complejos, integrando APIs, manejando rutas, y optimizando la experiencia del usuario.

React

Desarrollo interfaces de usuario interactivas con React, aplicando técnicas como el manejo eficiente del estado con Redux y optimizando la interacción con APIs RESTful. También he creado componentes reutilizables y aplicaciones single-page de alto rendimiento.

Java y Spring Boot

Utilizo Java y Spring Boot en la creación de microservicios robustos y escalables. Me especializo en la implementación de arquitecturas basadas en APIs RESTful, gestionando bases de datos y aplicando principios SOLID para mantener código limpio y mantenible.

Docker

Utilizo Docker para crear entornos de desarrollo y producción reproducibles. Creo contenedores ligeros que encapsulan aplicaciones y sus dependencias, facilitando la implementación y escalabilidad en la nube. También configuro redes y volúmenes para optimizar la estructura de los microservicios.

Google Cloud Platform (GCP)

Con experiencia en GCP, gestiono despliegues en la nube y la integración de servicios como Google Kubernetes Engine, Cloud Storage y Cloud Functions. También administro infraestructura en la nube utilizando Terraform y realizo implementaciones automatizadas.

Flask

Desarrollo aplicaciones backend ligeras con Flask, principalmente para servicios API REST. Utilizo Flask para crear prototipos rápidos y soluciones escalables, integrando bases de datos como MySQL o PostgreSQL y manejando autenticación de usuarios.

MySQL

Tengo experiencia en la creación y optimización de bases de datos relacionales con MySQL. He trabajado en la normalización de esquemas, optimización de consultas, y manejo de transacciones para asegurar la integridad de los datos en aplicaciones críticas.

AWS

Con experiencia en AWS, manejo despliegues en la nube, utilizando servicios como EC2, S3 y Lambda para desarrollar soluciones escalables. También monitoreo aplicaciones usando CloudWatch y configuro infraestructuras seguras y optimizadas.

Node.js

Desarrollo APIs backend utilizando Node.js, creando servicios rápidos y eficientes. Integro aplicaciones con bases de datos como MySQL y PostgreSQL, y manejo autenticación de usuarios y gestión de sesiones de manera segura.
